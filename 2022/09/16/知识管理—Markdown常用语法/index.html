知识管理—Markdown常用语法

发表于2022-09-16|更新于2022-09-16|知识管理

一、标题

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

二、标题

// 段落没有特殊的格式，直接编写文字就好
# 这是段落
// 段落后面如果超过一个空白字符（空格、换行符等）就会生成新的段落
# 这是段落`空格/回车键`
# 这是另一行段落

三、字体

**粗体文本**
_斜体文本_
**_粗斜体文本_**
~~带删除线的文本~~
<u>带下划线的文本</u>            # 通过HTML的<u>标签来实现
<kbd>Ctrl</kbd>+<kbd>C</kbd>   # Ctrl + C
{喜大普奔|xi da pu ben}        # 注音符号
[^脚注文本]                   # 脚注是可选功能，并非所有支持脚注的解析器都支持行内脚注

四、列表

// 无序列表，使用 - 带空格
- 第一项
- 第二项
- 第三项
// 有序列表，使用数字并加上 . 号来表示
1. 第一项
2. 第二项
3. 第三项
// 列表嵌套
1. 第一项：
    - 第一项嵌套的第一个元素
    - 第一项嵌套的第二个元素
2. 第二项：
    - 第二项嵌套的第一个元素
    - 第二项嵌套的第二个元素

五、区块

// 区块引用
> 标题
> 简介
// 区块嵌套
> 最外层
> > 第一层嵌套
> > > 第二层嵌套

六、代码

# 第一种方法：代码区块使用4个空格或者一个制表符（Tab键）
# 第二种方法：可以用 ``` 包裹代码，并指定一种语言（也可以不指定）

七、超链接

// 链接到网站
[链接名称](链接地址)  # 如:[百度以下](https://www.baidu.com)
// 无标签链接
<链接地址>           # 如:<https://www.baidu.com>
// 无标签邮箱链接
<123456@qq.com>
// 其他元素配合
[**粗体链接**](链接地址)  
[_斜体链接_](./Md页面链接地址)  
[`代码链接`](./Md页面中的代码块地址)

八、图片

// 图片语法格式
![图片简介](图片地址) # 如:[RUNOOB图标](http://static.runoob.com/images/runoob-logo.png)
// 横屏滑动幻灯片
<![图片简介](图片地址),![图片简介](图片地址)>
// Markdown不能设置图片的高度与宽度，可使用HTML的<img>标签
<img src="http://static.runoob.com/images/runoob-logo.png" width="50%">

九、表格

// 表格使用竖线(|)区分每列，在表格头和表格体之间使用三个减号(-)做分隔
|第一列|第二列|第三列|
|---|---|---|
|第一行第一列|第一行第二列|第一行第三列|
|第二行第一列|第二行第二列|第二行第三列|
// 设置表格对齐格式
|第一列|第二列|第三列|
|:---|:---:|---:|
|左对齐|居中对齐|右对齐|
// 在表格中使用其他元素
|第一列|第二列|  
|:---:|:---:|  
|**粗体**|_斜体_|  
|`行内代码`|[链接名称](链接地址)|
