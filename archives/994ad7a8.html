WebChatGPT开启联网功能+OpenAI API申请

发表于2024-01-19|更新于2024-01-19|ChatGPT

一、前言

使用ChatGPT的朋友都知道，该模型的对话知识库是基于OpenAI最近一次数据训练和建模日期的，无法提供截至到当前时间的最新信息。而WebChatGPT作为一款基于Chromium内核的浏览器插件，它可以让ChatGPT具备互联网访问权限。

  • 基于Chromium内核的浏览器均可使用WebChatGPT插件，包括Chrome、Edge和360极速浏览器等。
  • 提供最新、更准确的对话，让ChatGPT开启网络搜索，并分析搜索结果，以提供深度回答。
  • 有免费的一键ChatGPT提示库，包含数百条高质量的提示。
  • 自定义配置，根据偏好设置搜索过滤器、选择可信来源、设置隐私选项。

工作原理：ChatGPT操作栏输入问题 → WebChatGPT利用搜索引擎，将与问题最相关的结果返回给ChatGPT → ChatGPT归纳总结，最后生成回复

二、WebChatGPT安装（以Chrome浏览器为例）

1.访问chrome应用商店，并搜索 WebChatGPT

2.点击添加扩展程序并等待安装完成

3.插件设置简体中文

4.是否开启AI搜索

三、WebChatGPT选项设置

  • 1-click prompts：是否开启一键提示功能。
  • Web access：是否开启从搜索引擎中查找数据的功能。
  • Copilot：开启AI搜索更细致的网络查询功能。（适合难度高的问题）
  • Quick search：开启AI搜索更快捷的网络查询功能。（日常使用）
  • Prompt：该功能作用很简单，就是精确锁定问题的主题，避免ChatGPT跑题。（默认值是“default prompt”，选择该选项并不会时刻强调主题信息；如果设置prompt为“你是IT专家…”，那么每次当我们在操作栏输入问题，都会加上“你是IT专家…”）
  • Language：表示ChatGPT回复时所显示的语言
  • Region：根据所在地区（国家）查询符合条件的数据。（默认值是“Any region”：所有地区）
  • Time：从指定的时间范围内搜索匹配的文章。（默认值是“Any time”：所有时间）
  • Search results：显示搜索结果的参考来源。（凭个人喜好设置）
  • Search engine：选择所使用的搜索引擎。（默认为Google搜索）

四、OpenAI API

通过API调用模型进行收费，极其适合基于ChatGPT做二次开发的开发人员（如聊天机器人、语音识别等），而且在其他平台或软件上使用ChatGPT时，也经常需要提供API Key。API购买与开通会员类似，必须绑定国外信用卡，国内的Visa、Mastercard都无法支付。

1.OpenAI API使用

更多方法，可查阅官方文档：https://platform.openai.com/docs/overview

2.OpenAI 官网申请获取 API Key

url：https://platform.openai.com/api-keys

3.API Token收费标准
